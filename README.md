# News&Notes üìù

News&Notes is a simple pet-project for learning about such technologies:
* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [TypeScript](http://www.typescriptlang.org/)
* [Webpack](https://webpack.js.org/)
* [Materialize](https://materializecss.com/)

It could be run both in desktop and mobile browsers.

## Installation üíª
Clone this repository to computer.

```bash
git clone https://github.com/aakond/news-and-notes.git
```

Change the directory to the root project directory.

```bash
cd news-and-notes
```

Use the package manager [npm](https://www.npmjs.com/get-npm) to install packages.

```bash
npm install
```


Start app in development mode with the next command.

```bash
npm start
```

Open [http://localhost:8080](http://localhost:8080.).
